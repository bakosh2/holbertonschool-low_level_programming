# C - Variables, if, else, while

This project contains C programs that demonstrate the use of variables, conditional statements (if, else), and while loops.

## Projects

- **0-positive_or_negative.c**: Determines if a random number is positive, negative, or zero.
- **1-last_digit.c**: Prints the last digit of a random number and provides information about it.

## Learning Objectives

- Understand variables and data types in C
- Use if, else if, and else statements
- Work with conditional logic
- Generate random numbers
- Use arithmetic operators

## Requirements

- Compiled on Ubuntu 20.04 LTS using gcc
- Betty coding style compliant
- No errors or warnings during compilation
