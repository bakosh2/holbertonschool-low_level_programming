0-print_dlistint.c  Function Description
This function prints all elements of a dlistint_t doubly linked list.

Prototype
size_t print_dlistint(const dlistint_t *h);

How it works:

Starts from the head of the doubly linked list.
Iterates forward using the next pointer.
Prints the integer value n stored in each node.
Counts how many nodes were printed.
Returns the total number of nodes.

What this task demonstrates:

Understanding of a doubly linked list structure.
Ability to traverse a linked list.
Working with pointers and struct navigation.
Following Holberton C coding style.
